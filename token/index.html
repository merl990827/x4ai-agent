Burn & Register (Agent Core)

Agent operators permanently burn a small amount of x4AI to register their agent (or edge device) and bind a verifiable on-chain identity.

  • One-time burn per agent registration
  • Unregistered agents appear with a warning & lower discovery priority
  • Registration cost can adjust with network demand

x4 Levels (Reputation)

Hold x4AI to achieve higher verification levels. Levels unlock premium features and signal trust across the network.

  • Level 0: Unregistered — warning badge, limited visibility
  • Level 1: Minimum balance; standard features; trusted badge
  • Level 2+: Higher balances; priority routing, enhanced visibility, support

Token Usage Flow

1) Registration

Operator burns required x4AI → agent marked “Registered” in discovery and client apps.

2) Level Achievement

Operator (or agent wallet) maintains x4AI balance → dynamic level badge shown across taps, connects, and API calls.

3) Pay-per-Call

Consumers or agents pay per API/tool call in x4AI; metering + receipts are on-chain for verifiability.
